HTML Table Elements

Tag Tag Name Description
<table> Table Creates and wraps a table.
<tr> Table Row Indicates the start of a row.
<td> Table Data Adds a cell of data to a row.
<th> Table Heading Represents the heading of a column or row.
<thead> Table Header Wraps the headings of the columns.
<tbody> Table Body Wraps the actual table data.
<tfoot> Table Footer Wraps the footer of the table (e.g. sum)

HTML Table Attributes

Attribute Attribute Name Description
colspan Column Span Indicates how many columns the cell should run across.
rowspan Row Span Indicates how many rows the cell should run across.
scope Scope Indicates whether a heading is a column or a row.
